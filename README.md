# Cerberus BOT
Discord bot working on discord.js v.13 
## Installation

```bash
# For auto-refesh dev:
npm i -g nodemon

npm install
```
- Create .env file at ./src
- Modify ./src/config.js

**.env**:
```env
BOT_TOKEN=[Paste Your Token Here]
```

## Usage

```bash
# Normal start
npm run start

# Developing
npm run dev
```
